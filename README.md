## Overview

This repository hosts a backend Web API built with **NestJS** and **TypeScript**.  
The project is designed as part of a learning and practice workflow, focusing on clean code structure, dependency injection, and RESTful API development.

The codebase follows a modular approach to ensure clarity, extensibility, and ease of maintenance as the application grows.

---

## Key Features

- Modular and layered architecture  
- RESTful API design  
- TypeScript for type safety and clarity  
- Dependency Injection using NestJS core mechanisms  
- Easily extensible for future features and integrations  

---

## Technology Stack

- **Node.js** – JavaScript runtime  
- **NestJS** – Backend framework  
- **TypeScript** – Strongly typed superset of JavaScript  
- **pnpm** – Package manager
